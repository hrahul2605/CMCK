## Running the application

- From Docker image
```bash
    # pull the docker image
    $ docker pull ghcr.io/hrahul2605/cmck:latest

    # run the docker image on PORT
    $ docker run -p {PORT}:5000 ghcr.io/hrahul2605/cmck:latest
```
